# South-Wales-housing-analysis
An independent data analysis project examining what drives 
house prices across South Wales using 135,000+ Land Registry 
transactions (2020–2025).

## Key Findings
- Deprivation accounts for a £200,000+ price gap
- Property type is the single strongest price driver
- Cardiff and Monmouthshire command the largest location premiums

## Tools Used
R, ggplot2, tidyverse, Land Registry, WIMD, ONS

## Data
The raw datasets are not included in this repo due to file size. 
They can be downloaded from:
- Land Registry: gov.uk/government/statistical-data-sets/price-paid-data-downloads
- WIMD 2019: statswales.gov.wales
- ONS Postcode Lookup: geoportal.statistics.gov.uk
